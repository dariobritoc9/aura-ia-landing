# Aura IA - Landing Page Premium

Landing page moderna y sofisticada para Aura IA, una agencia especializada en automatizacion e inteligencia artificial para PYMES y emprendedores.

## Caracteristicas
Diseno Glassmorphism: Estetica moderna con efectos de cristal y desenfoque de fondo.
Tema Oscuro: Paleta de colores premium optimizada para una apariencia tecnologica.
Totalmente Responsivo: Adaptado para dispositivos moviles, tablets y escritorio.
Interaccion Fluida: Animaciones de revelacion al hacer scroll y navegacion suave.
Iconografia Moderna: Uso de Lucide Icons para una identidad visual clara.

## Tecnologias Utilizadas
HTML5: Estructura semantica y accesible.
CSS3: Variables (Custom Properties), Flexbox, CSS Grid y efectos de backdrop-filter.
JavaScript: Logica de scroll, animaciones y manejo de interaccion del usuario.
IA: Imagen hero generada especificamente para combinar con la estetica del sitio.

## Estructura del Proyecto
/
|-- index.html      # Estructura principal
|-- style.css       # Estilos y diseno responsivo
|-- script.js       # Interactividad y animaciones
L-- hero.png        # Imagen hero generada por IA
